PROJECTS

Wedding Seating Arrangement

A program that outputs a wedding seating arrangement based on the number of guests, number of tables, and number of seats per table. In addition, it takes into account people who must sit with each other, and people who cannot sit with each other. This is done by implementing the DPLL algorithm and using SAT encoding, written in Python.

Bucket Filter System

A bucket filter system that takes incoming packets and services them based on their token value This is done via multithreading to resemble a bucket filter whose components work simultaneously, written in C.

Maze Search

A maze finder that takes in a maze, starting point, and ending point, and finds a valid path from the starting to ending position. It will state if a path is not possible. This is implemented with recursion using Java.

Google Earth Spirograph

A KML file that has coordinates of locations close to me. The coordinates to make a convex hull and spirograph were found using Postgres queries on Google Cloud, and were run on Google Earth.
